# Debug Node programm
```jsx
// Node debugging commands
// To begin a node program normally
node path/to/myProgram.js

// To run the program in debug mode
node inspect path/to/myProgram.js

// Game commands to move around in the computer program world
cont, c: Continue execution
next, n: Step next
step, s: Step in
out, o: Step out
pause: Pause running code (like pause button in Developer Tools)

```
