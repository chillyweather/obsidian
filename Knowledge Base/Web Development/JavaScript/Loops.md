# Loops

### While
```js
while (condition) {
    // body
}
```

```js
let number = 10;

while (number <= 20) {
    console.log(number);
    number += 2;
}
```



### For
```js
for (initialization; condition; increment) {
    // the loop body
}
```

```js
for (let i = 0; i <= 10; i++) {
  console.log(i);
}
```

### Do ... While
```js
do {
    // the loop body
} while (condition);
```

```js
let number = 21;

do {
    console.log(number);
} while (number <= 20);
```
