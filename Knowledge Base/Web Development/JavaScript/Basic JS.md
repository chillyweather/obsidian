### Template literals
```js

```

### ## Logical NOT, AND, and OR

#### Not
```js
let merry = true;
console.log(!merry); // false

let christmas = false;
console.log(!christmas); // true
```

#### And
```js
let merry = true;
let christmas = false;

console.log(merry && christmas); // false
```

#### Or
```js
let merry = true;
let christmas = false;

console.log(merry || christmas); // true
```
